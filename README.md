# TaskTracker

A task tracking iOS app built with SwiftUI and SwiftData.

## Features

- Add new tasks
- Mark tasks as complete with a tap
- Swipe to delete tasks
- Data persistence across app launches (SwiftData)

## Tech Stack

- Swift 6
- SwiftUI
- SwiftData
- Xcode 16+

## Requirements

- macOS with Xcode 16 or later
- iOS 17.0+ simulator or device

## How to Run

1. Clone or download this repository
2. Open `TaskTracker.xcodeproj` in Xcode
3. Select a simulator or connected device
4. Press `Cmd + R` to build and run
